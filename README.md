🌿 Plant Disease Detection

📌 Overview
This project is a deep learning-based web application that detects rice leaf diseases from images. 
The system takes a leaf image as input and predicts the disease type using a trained CNN model. 
The goal is to help in early disease identification using Artificial Intelligence.

🛠 Technologies Used
- Python
- PyTorch
- ResNet-18 (Transfer Learning)
- NumPy
- Pandas
- Matplotlib
- Scikit-learn
- Flask

🔄 Project Workflow
1. Collected and prepared rice leaf image dataset  
2. Performed image preprocessing (resizing, normalization)  
3. Applied data augmentation (rotation, flipping)  
4. Used transfer learning with ResNet-18  
5. Trained and evaluated the model  
6. Deployed the model using Flask web application  

⚙️ How to Run the Project

Install required libraries:

pip install -r requirements.txt

Run the application:

python app.py

🎯 Features
- Image-based disease prediction
- Deep learning classification model
- Real-time prediction using Flask
- Simple and user-friendly interface
